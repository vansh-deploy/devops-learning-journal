# Linux Commands Reference

## Week 1 — Filesystem & Navigation

### ls -la
Lists all files including hidden ones with permissions, owner, size, date.
- `-l` = long format (shows permissions)
- `-a` = all files including hidden (files starting with .)
- Example: `ls -la /etc/nginx/`

### pwd
Print Working Directory — shows your current location in the filesystem.
- I use this when I'm lost and need to confirm where I am.

### cd
Change Directory.
- `cd /etc` — go to /etc
- `cd ..` — go up one level
- `cd ~` — go to home directory
- `cd -` — go back to previous directory (very useful)
